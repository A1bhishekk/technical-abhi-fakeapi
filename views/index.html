Welcome To Technical Abhi API Services

This page will list all of the rest services. These are fake online REST APIs for testing and prototyping sample applications that use rest calls to display listings and crud features.

There are following public apis :

S.N Route Method Type Full Route Description
1 /users GET JSON https://muddy-fox-scarf.cyclic.app/users Get all user details
2 /users/id GET JSON http://localhost:5000/users/id Get single user details
3 /register POST JSON https://muddy-fox-scarf.cyclic.app/register Create new record in database
4 /update/id PUT JSON https://muddy-fox-scarf.cyclic.app/update/id Update a user record
5 /delete/id DELETE JSON https://muddy-fox-scarf.cyclic.app/delete/id Delete a user record
